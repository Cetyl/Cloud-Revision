# Private Enterprise AI Chatbot using GCP + Gemini

## Overview

This project provides a fully private, enterprise-grade AI chatbot similar to ChatGPT using Google Cloud Platform (GCP) and Gemini models through Vertex AI.

The chatbot is designed for internal company usage only and is not exposed publicly.

The solution supports:

* Private internal access
* Google Gemini models via Vertex AI
* Enterprise authentication
* Retrieval-Augmented Generation (RAG)
* Internal document search
* Terraform infrastructure deployment
* Scalable containerized workloads
* Secure networking
* Audit logging and monitoring

---

# Architecture

```text
Employees
   ↓
Internal HTTPS Load Balancer
   ↓

   ↓
FastAPI Backend
   ↓

   ↓

```

## Example FastAPI Backend

```python
from fastapi import FastAPI
from vertexai.generative_models import GenerativeModel
import vertexai

app = FastAPI()

vertexai.init(project="company-private-gpt", location="us-central1")

model = GenerativeModel("gemini-2.5-pro")

@app.post("/chat")
async def chat(prompt: str):
    response = model.generate_content(prompt)
    return {
        "response": response.text
    }
```

# Step 8 — Containerize Application

## Dockerfile

```dockerfile

WORKDIR /app

COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .

CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8080"]
```

# Step 19 — CI/CD with GitHub Actions

## Example Workflow

```yaml
name: Deploy

on:
  push:
    branches:
      - main

jobs:
  deploy:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - uses: google-github-actions/auth@v2
        with:
          credentials_json: ${{ secrets.GCP_SA_KEY }}

      - uses: google-github-actions/setup-gcloud@v2

      - run: |
          docker build -t chatbot-api .
```
